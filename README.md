# Socket_chatting
MFC기반 Socket Communication을 이용한 양방향 데이터 통신 프로젝트

개요
===
MFC를 이용하여 간단한 Socket 통신 프로그램에  
+ TCP/UDP이용한 Full-Duplex Multi-thread 
+ 사진 전송
+ Timeout
+ ARQ ( REJ, ACK, etc.. )
+ Checksum
+ Piggyback
+ Sliding Window
+ Segmentation ~ Reassemble  

기능 추가하여 데이터통신의 개념 실습.
